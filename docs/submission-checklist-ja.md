## E. 実機スモーク（提出直前）

新規インストール推奨（初回ヒント確認用）。

### 起動・設定
- [ ] 初回のみ「明るさ」「言語」の説明シートが出る（既存端末では出ない）
- [ ] 右上でライト/ダーク、EN/JP が切り替わる
- [ ] ℹ️ → クレジット・出典が開く
- [ ] クレジット内のプライバシー / 利用規約リンクがブラウザで開く
- [ ] お問い合わせ（mailto）が起動する

### 地図・島
- [ ] 日本地図で各地域ピンが選べる（忽那と小豆・直島が重なりすぎない）
- [ ] 島詳細ヘッダー・背景・クレジットが表示される

### 天気
- [ ] 天気が取得できる（またはキャッシュ／失敗文言）
- [ ] 波の高さ（取得できる島）
- [ ] Open-Meteo 出典リンク
- [ ] 「○分前に取得…」などキャッシュ文言（2回目以降）

### ダイヤ
- [ ] 船便見出しにフェリーアイコン＋青系
- [ ] 航空便がある島で飛行機アイコン＋空色（例: 佐渡・与那国）
- [ ] 欠航バナーから公式サイトへ遷移できる
- [ ] 八重山以外は代表ダイヤ／公式リンクである旨が分かる

### 店舗・写真・位置
- [ ] 店舗タブで Apple マップ結果が出る（または圏外失敗）
- [ ] 位置情報許可ダイアログの文言が用途どおり
- [ ] 写真メモ: カメラ／ライブラリ許可文言が用途どおり
- [ ] 写真が端末内に保存され、アプリ削除で消える想定である

### オフライン（可能なら）
- [ ] 機内モードで、以前開いた島はキャッシュ表示
- [ ] 未取得データは短時間で失敗表示になる

## よくある落とし穴

1. 規約 URL を古い `opaquu.github.io` のまま Connect に入れる → 404 でリジェクトリスク  
2. 説明文に WBGT を残す → 機能と不一致  
3. 英語モードのスクショを使う場合、一部日本語が残ることを把握する（全面バイリンガルではない）  
4. ローカルだけの未コミット変更で Archive し、リモートと食い違う
